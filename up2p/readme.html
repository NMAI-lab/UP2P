Welcome to U-P2P! Universal Peer-to-Peer is a framework for sharing files and discovering new communities in a peer-to-peer network.

To get started with U-P2P, consult the Universal Peer-to-Peer User's Guide.

To jump right in, launch U-P2P by running the startup script in the Apache Tomcat Bin directory and open a web browser to http://localhost:8080/up2p.

Project Contents

Known Issues

The following issues are known to occur in U-P2P:

  • When removing resources from a community a refresh of the page may be required for changes to appear.
  • Deleting a community without deleting all its contained resources beforehand can cause errors if the community is recreated on the same node. This is due to a caching issue, and can be fixed by restarting the U-P2P node.
  • Shutting down Tomcat is sometimes (always!) prevented by a running instance of U-P2P. Although Tomcat may claim that it has shutdown, it is still resident in memory. This prevents U-P2P from initializing properly if it is restarted. Workaround: After running the shutdown script, manually kill Tomcat's java process.
